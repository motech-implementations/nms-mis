Kilkari Line Listing Report

Listened to < 25% this month


This report gives the line-listing of all the beneficiaries who have been listening to an average of less than 25% of the content in the last calendar month in the Kilkari system as observed in the selected month. This report tracks beneficiaries who have either zero successful calls or not listened complete message (i.e., on an average of <25%). This report should not include ‘completed’ or ‘deactivated’ beneficiaries.

This line listing report would be available for download for the respective state, district and block users. These reports would be e-mailed to the respective block and district users periodically once it is generated. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day.

The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries to listen to the Kilkari content to the full duration of 90 secs. If the mobile number registered against that beneficiary does not belong to the beneficiary or that number is not accessible to them during normal calling hours, then the beneficiary should be advised to provide the correct mobile number or provide an alternative number that would be accessible to the beneficiary more frequently.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Select "Kilkari Reports" under "Reports Category"




3) Then select "Listen to < 25% this month" under "Reports"




4) Then select the Month for which the reports are needed




5) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




6) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




7) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




8) To view the result, click the "Submit" button




9) Click on the download link to download the report. The selected report will be downloaded in Excel files.




10) Clicking on ‘Reset’ button will reset all the fields to their default values.

Deactivated for Low Listenership

This report gives the line-listing of all the beneficiaries who have been deactivated in the selected month by the Kilkari system for listening less than 25% of the content for each successful call spread over six consecutive weeks.

Both the line listing report would be available for download for the respective state, district and block users. These reports would be e-mailed to the respective block and district users periodically once it is generated. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day.

The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries and understand why they are not listening to the Kilkari calls. If the mobile number belongs to the correct beneficiary then they should be motivated to reactivate their Kilkari Subscription and listen to the Kilkari content for the complete duration of 90 secs. If the mobile number registered against that beneficiary does not belong to the beneficiary or that number is not accessible to them under normal calling hours, then the beneficiary should be advised to provide the correct mobile number or provide an alternative number that would be accessible to the beneficiary more frequently.

This report will provide the information about the beneficiaries who have been deactivated in the selected month for listening to less the 25% of Kilkari messages for a consecutive period of 6 weeks.

Steps to download the reports

4) Then select the Month for which the reports are needed

Mother Rejected Records

This report gives the line-listing of all Mother beneficiaries whose registered mobile numbers are either incorrect or are duplicate. These mobile numbers had flown in to the IVRS system but were rejected by the acceptance logic run by the IVRS system. The report contains the MCTS or RCH IDs. This report would be generated once every week preferably on Sunday. This line listing report would be available for download for the respective district and block users. These reports would be e-mailed to the respective block and district users on the day after it was generated –on Tuesday. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day. The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries and register their correct mobile numbers in the RCH Application. In this way we can correct the mobile numbers of beneficiaries and bring a higher percentage of beneficiaries under the ambit of Kilkari program. This report is a very useful actionable report for the State / UT to correct the beneficiary contact mobile number.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Select "Kilkari Reports" under "Reports Category"




3) Then select "Mother Rejected Records" under "Reports"




4) Then select the Month under "Select Week"




5) Then select the 1st day of week(Sunday) for which the report is needed under "Select Week"




6) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




7) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




8) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




9) To view the result, click the "Submit" button




10) Click on the download link to download the report. The selected report will be downloaded in Excel files.




11) Clicking on ‘Reset’ button will reset all the fields to their default values.

This report tracks beneficiaries who have either zero successful calls or not listened complete message (i.e., on an average of <25%). This report should not include ‘completed’ or ‘deactivated’ beneficiaries.

This line listing report would be available for download for the respective state, district and block users. These reports would be e-mailed to the respective block and district users periodically once it is generated. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day.

The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries to listen to the Kilkari content to the full duration of 90 secs. If the mobile number registered against that beneficiary does not belong to the beneficiary or that number is not accessible to them during normal calling hours, then the beneficiary should be advised to provide the correct mobile number or provide an alternative number that would be accessible to the beneficiary more frequently.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Select "Kilkari Reports" under "Reports Category"




3) Then select "Listen to < 25% this month" under "Reports"




4) Then select the Month for which the reports are needed




5) Since this is for block user, location details upto “Block” are already selected.




6) To view the result, click the "Submit" button




7) Click on the download link to download the report. The selected report will be downloaded in Excel files.




8) Clicking on ‘Reset’ button will reset all the fields to their default values.

Mother Rejected Records

This report gives the line-listing of all Mother beneficiaries whose registered mobile numbers are either incorrect or are duplicate. These mobile numbers had flown in to the IVRS system but were rejected by the acceptance logic run by the IVRS system. The report contains the MCTS or RCH IDs. This report would be generated once every week preferably on Sunday. This line listing report would be available for download for the respective district and block users. These reports would be e-mailed to the respective block and district users on the day after it was generated –on Tuesday. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day. The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries and register their correct mobile numbers in the RCH Application. In this way we can correct the mobile numbers of beneficiaries and bring a higher percentage of beneficiaries under the ambit of Kilkari program. This report is a very useful actionable report for the State / UT to correct the beneficiary contact mobile number.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Select "Kilkari Reports" under "Reports Category"




3) Then select "Mother Rejected Records" under "Reports"




4) Then select the Month under "Select Week"




5) Then select the 1st day of week(Sunday) for which the report is needed under "Select Week"




6) Since this is for block user, location details upto “Block” are already selected.




7) To view the result, click the "Submit" button




8) Click on the download link to download the report. The selected report will be downloaded in Excel files.




9) Clicking on ‘Reset’ button will reset all the fields to their default values.
