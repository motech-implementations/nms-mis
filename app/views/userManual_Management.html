User Management


The User Management tab allows Admin users to view/create new lower/equal access-level users and lower access-level admins, and to edit/delete users created by them.

1) User Management can be used by admin users only after logging in and then clicking on the 'User Management' tab in the navigation bar:




2) Users-list containing all admins and users is shown to National Admin:




Functionalities in User Management page:

3) A search field is provided to search for entered keywords from all fields of all the records:




4) Filters for 'Access-type', 'Access-level', 'State', 'District' and 'Block' are provided, which can be used to filter the respective columns.

5) The first five columns can be sorted by clicking on the column headers.




6) Each page shows a maximum of 10 users. Use the pagination at the bottom-right to navigate to other pages:




7) National Admin can create Admins (other than National-admins) and any Users by clicking on the 'Create new User' button on the top-right of the screen.

Multiple Admins cannot be created for any State or District

The following screen is shown with empty fields which are required to be filled.

All fields in the form are compulsory. Default password for creating a user is the user's contact number.

Create user form has two action buttons: 'Create user' and 'Cancel'.

  • Create new User: creates a new user with details as filled in the form.
  • Cancel: aborts the action of creating user and navigates back to User Management.



8) Admins can only edit users they created, by clicking on the 'Edit' button. Every user on the list maynot have this button:




9) 'Edit' will show the following screen on click.

'Edit' user form is auto-filled with previous information of the user.




10)Edit form has four submit options:

  • Update user: This option will validate all the values entered and update the same.
  • Reset password: Will reset the password of the user to their contact number.
  • Delete user: Will delete the user. Only those users who have never logged into the application can be deleted.
  • Cancel: Cancels edit action.


11) 'Upload Bulk Users' button at the top-right of the User Management page navigates to the bulk upload page.

In this page, the admin can download sample template.

The admin can then fill details in a CSV format according to given headers in the template and upload it back to the same page:



After uploading the file, the admin must click the 'Upload Bulk Users' button. The file is then validated.



If the validation is successful, users will have their account created otherwise validation errors will then be displayed after processing the file.
