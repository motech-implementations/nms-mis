User Management


The User Management tab allows Admin-access users to view, create new lower/equal level users, and to edit/delete users created by them

1)To use the User Management section, follow the steps in the User Manual - Profile Section to login

Only Admin level users can access User Management by clicking on the 'User Management' tab:




2)Click on 'User Management' to display user-list according to your access level:




Functionalities in User Management page:

3)A textbox is provided to search for through all fields of all the records by entering keywords:




4)Pre-determined filters Access-type, Access-level, State, District and Block can be used to filter information:




5)Each page shows 10 users. Use the buttons on the bottom-right to access other pages of user-information:




6)Admins can only edit users they created by clicking on the 'Edit' button. Every user on the list maynot have this button:




7)'Edit' button and 'Create new User' button will both show the following screen on click:




8)'Create new user' will show empty fields, while 'Edit' will fill fields with previous information

All fields in the above form are compulsory. Default password for creating a user is the user's contact number.

This form has three submit options:

  • Update user: This option will validate all the values entered and update the same.
  • Reset password: Will reset the password of the user to their contact number and update the contact number as well.
  • Delete user: Will delete the user.


9)'Bulk Upload Users' button at the top-right of the User Management page leads to the bulk upload page:

In this page, the admin can download sample template with all required user details to be uploaded.

The validation errors will then be displayed after processing the file.
