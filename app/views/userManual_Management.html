User Management


The User Management tab allows Admin users to view, create new lower/equal access-level users, lower access-level admins and to edit/delete users created by them.

1)To access the User Management page, login with your admin credentials.

User Management tab is visible to admin users only after logging in to the application

Click on the 'User Management' tab:




2)Click on 'User Management' tab to display user-list according to your access level:




Functionalities in User Management page:

3)A search field is provided to search for entered keywords from all fields of all the records:




4)Filters 'Access-type', 'Access-level', 'State', 'District' and 'Block' are provided with certain values,which can be used to filter the respective columns.

5)The first five columns can be sorted by clicking on the headings/names of these columns.




6)Each page shows a maximum of 10 users. Use the numbers and arrows on the bottom-right to navigate to other pages:




7)Admins can only edit users they created, by clicking on the 'Edit' button. Every user on the list maynot have this button:




8)Admins can create other admins of lower access-level or equal/lower access-level users by clicking on the 'Create new User' button.

9)'Edit' button and 'Create new User' button will both show the following screen on click:




10)'Create new User' button will show empty fields with the Create User header, while 'Edit' will fill fields with previous information on the user.

All fields in the above form are compulsory. Default password for creating a user is the user's contact number.

Create user form has two action buttons: 'Create user' and 'Cancel'.

Edit form has four submit options:

  • Update user: This option will validate all the values entered and update the same.
  • Reset password: Will reset the password of the user to their contact number and update the contact number as well.
  • Delete user: Will delete the user. Only those users who have never logged into the application can be deleted.
  • Cancel: Cancels edit action.


11)'Upload Bulk Users' button at the top-right of the User Management page leads to the bulk upload page.

In this page, the admin can download sample template.

The admin can then fill details in a CSV form according to given headers in the template and upload it back to the same page:



After uploading the file, the admin must click the 'Upload Bulk Users' button. The file is then validated.



The validation errors will then be displayed after processing the file.
