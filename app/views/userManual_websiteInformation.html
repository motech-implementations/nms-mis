FAQ

The FAQ page contains answers to frequently asked questions about the MIS portal.

1) The FAQ page can be accessed by clicking on FAQ tab:



2) The FAQ landing page shows the list of high level FAQ subjects on the left pane (as highlighted on the snapshot below) The General Information is the default selected tab.

3) Clicking on any of the tabs on the left pane navigates to the respective section.

4) Each section has frequently asked questions categorised and collated for that subject.

5) On clicking each of the FAQ tabs, a list of available questions on the subject, is displayed. Click on any question to see the answer to that question.

Forgot password

1) Use the forgot password link provided on the homepage (highlighted on the snapshot below) in case if you have forgotten your password



2) In case of forgot password user will click on the forgot password link to open forgot password page.



3) Enter username in the form and click the captcha check box and click submit. Once username is identified the system will send a new system generated password to user’s email id. A notification as shown in the screenshot below will be shown.



4) When user will use the new password for the first time system will instruct user to change the password (force change) as shown in below screenshot.



5) After submitting the form password will be changed successfully and a notification as shown in the screenshot below will be shown .
