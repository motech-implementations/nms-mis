FAQ

The FAQ page contains some frequently asked questions about the various sections of the application.

1) The FAQ page can be accessed by anyone by clicking on the FAQ link:



2) In the FAQ page, the Website Information link is the default selected tab.

3) Clicking on any of the tabs in the sidebar will take the user to the respective section.

4) In each section, some frequently asked questions are mentioned.

5) Clicking on a question will show the answer to that question under it.

6) Clicking on the question again will hide the answer. This is compulsory to hide the answer:




Feedback Form and Contact Us Page

1) The Feedback tab is provided at the top-right of any page. Clicking on this tab will show a feedback form:



2) A user who intends to provide feedback on the experience and improvements towards the application must provide all details marked with an asterisk(*).




3) After filling up the form, the user must click on the Captcha tick-box and click on submit to send their feedback after a tick-mark appears.

4) This will show a message regarding success/failure. In case of a successful submission, the user is navigated to a success page, with a link to return to the feedback page.



5) The Contact Us link beside the Feedback link navigates the user to the contact page:



6) This page shows the user a number of contact numbers, email addresses and location which can be used to contact the Ministry of Health

7) A request for contact/message can also be sent by the user using the given info by submitting the contact form using the same process as the feedback form.

8) Users must compulsorily fill the fields of the contact form with their own details, check the captcha tick-box and submit the form, when necessary.



At the bottom-right of all pages, information regarding the latest update of the application, the total number of visits to all pages and the date and time of the latest loading of the current page is provided in the footer
