FAQ

The FAQ page contains some frequently asked questions about the various sections of the application

The FAQ page can be accessed by anyone by clicking on the FAQ link:



In the FAQ page, the Website Information link is pre-selected.

Clicking on any of the links in the sidebar will take the user to the respective section.

In each section, some frequently asked questions are mentioned as links.

Clicking on the link to a question will show the answer to that question under it.

Clicking on the question link again will hide the answer. this is compulsory to hide the answer:



Feedback Form

The Feedback link is provided at the top-right of any page. Clicking on this link will show a feedback form:



A user who intends to provide feedback on the experience and improvements towards the application must provide all details marked with an asterisk(*)

After filling up the form, the user must fulfil the Captcha condition and click on submit to send their feedback

This will show a message regarding success/failure. In case of a successful submission, the form is reset to blank fields



Contact Page

The Contact Us link beside the Feedback link takes the user to the contact page:



This page shows the user a number of contact numbers, email addresses and location which can be used to contact the Ministry of Health

A request for contact/message can also be sent by the user using the given info by submitting the contact form

Users must compulsorily fill the fields of the contact form with their own details and submit the form, when necessary.



At the bottom-right of all pages, information regarding the latest update of the application, the total number of visits to all pages and the date and time of the latest loading of the current page is provided in the footer
