FAQ

The FAQ page contains answers to frequently asked questions about the MIS portal.

1) The FAQ page can be accessed by clicking on FAQ tab:



The FAQ landing page shows the list of high level FAQ subjects on the left pane (as highlighted on the snapshot below) The General Information is the default selected tab.

3) Clicking on any of the tabs on the left pane navigates to the respective section.

4) Each section has frequently asked questions categorised and collated for that subject.

5) On clicking each of the FAQ tabs, a list of available questions on the subject, is displayed. Click on any question to see the answer to that question.

1) Clicking on the Feedbackprovided at the top-right corner (as highlighted on the snapshot below) shows a feedback form:



2) This section allows a user to provide any feedback a user might want to convey on the experience on the MIS portal, as well as suggest any improvements. A user needs to fill in all mandatory details, marked with an asterisk( * ),
Enter the correct captcha value and submit


3) A notification confirming successful submission of feedback will be shown

5) This page displays information on contact numbers, email addresses and locations of the 'Ministry of Health and Family Welfare', for offline communication with the Ministry.

7) Fill the contactUs form with all mandatory details marked with an asterisk( * ),
Enter the correct captcha value and submit



Forgot password

1) Use the forgot password link provided on the homepage (highlighted on the snapshot below) in case if you have forgotten your password



2) In case of forget password, user will click on reset password, system will ask for his/her username or email ID with captcha and once username or email ID is identified the system will send a new system generated password to user’s email id and when user will use the new password for the first time system will instruct user to change the password (force change)



3) Fill the from by following the given conditions



4) After filling the form click the submit form



5) After submitting password will be changed successfully
