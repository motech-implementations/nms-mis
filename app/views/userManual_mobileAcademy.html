Mobile Academy Reports

Cumulative Course Completion


This report is generated from the launch of service in the state till the end of selected month

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/app/.

2) Under "Reports Category" select "Mobile Academy Reports", then





3) Under "Reports" select "Cumulative Course Completion"





4) Then select the Month





5) Then select respective State, District and Block










5) Since this is for block user, location details from “Selected Block” are already selected.




5) Since this is for District user/admin, Select any/all block from “Selected Block”, under specified "District Block" and "State Block".




5) For State users, select any/all block from “Selected Block”, under any district from “District Block” with specified "State Block".







6) To view the result, press the "Select" button, or to reset it, press "Reset" button



7) Click on the download link to download the report. All the reports will be downloaded as Excel files.


8) Clicking on ‘Reset’ button will reset all the fields to their default values.


Circle wise Anonymous Users


This is a monthly report which lists out all the Mobile numbers that called from a specific circle in the selected month.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/app/.

2) Under "Reports Category" select "Mobile Academy Reports", then





3) Under "Reports" select "Circle wise Anonymous Users"





4) Then select the Month





5) Then select respective Circle from "Selected Circle"




5) Since this is for block user, location details from “Selected Circle” is already selected.




5) Since this is for District user/admin, location details from “Selected Circle” is already selected.




5) For State users/admin. location details from “Selected Circle” are selected from that state.




6) To view the result, press the "Select" button, or to reset it, press "Reset" button



7) Click on the download link to download the report. All the reports will be downloaded as Excel files.


8) Clicking on ‘Reset’ button will reset all the fields to their default values.
