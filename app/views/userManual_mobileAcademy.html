Mobile Academy Line Listing Report

Course Completion


This report is generated from the launch of service in the state till the end of selected month

This report gives the line-listing of all the ASHAs who have successfully completed the course for the very first time in the selected month, with details such as - when an ASHA started the course and when she completed the course. It has following columns:

  1. S. No.
  2. ASHA Name – name of the ASHA
  3. ASHA MCTS/RCH ID – ASHA ID as received from MCTS/RCH
  4. Mobile Number – ASHA’s mobile number as received from MCTS/RCH
  5. State – ASHA’s state as received from MCTS/RCH
  6. District – ASHA’s district as received from MCTS/RCH
  7. Taluka – ASHA’s taluka as received from MCTS/RCH
  8. Health Block – ASHA’s health block as received from MCTS/RCH
  9. Health Facility – ASHA’s health facility as received from MCTS/RCH
  10. Health Sub Facility – ASHA’s health sub facility as received from MCTS/RCH
  11. Village – AHSA’s Village as received from MCTS/RCH
  12. ASHA Creation Date – this is the date when ASHA records came in to the Mobile Academy system for the first time
  13. ASHA Job Status – ASHA’s GF Status as received from MCTS/RCH
  14. First Completion Date – the date when ASHA’s successfully completed the Mobile Academy course for the first time
  15. SMS Sent Notification – delivery status of the SMS sent to ASHA on successful completion. ‘True’ means the system received the delivery notification and ‘False’ means the system didn’t receive the delivery notification due to network issue.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Under "Reports Category" select "Mobile Academy Reports", then




3) Under "Reports" select "Course Completion"




4) Then select the Month for which the reports are needed




5) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




6) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




7) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




8) To view the result, click the "Submit" button.




9) Click on the download link to download the report. The selected report will be downloaded in Excel files.




10) Clicking on ‘Reset’ button will reset all the fields to their default values.


Circle wise Anonymous Users


This report gives a line-listing of all such users (i.e. mobile numbers) who had attempted the Mobile Academy toll-free number but these calls got rejected by the system as the users were not registered in the system. It is presumed that these anonymous numbers are the numbers that are used by ASHAs but are not registered with MCTS/RCH. The States / UTs can act upon this report and call up these numbers and after verifying that these numbers indeed belong to the working ASHAs should advise ASHA to get their numbers registered in the MCTS/RCH application. In this way, the application can assist States / UTs to register correct mobile numbers of ASHAs and bring all the ASHAs under the ambit of Mobile Academy program. It has following columns:

  1. S. No.
  2. Circle name – this is the telecom circle name as defined by TRAI (Telecom Regulatory Authority of India), because this user is not registered with the backend system, no profile information is available, therefore the system cannot provide the state name.
    Please note the telecom circles that cover the following state
    1. Assam – covers the state of Assam
    2. Bihar – covers the states of Bihar & Jharkhand
    3. Delhi – covers Delhi NCR including Gurugram, Noida, Ghaziabad & Faridabad
    4. Haryana – covers the state of Haryana
    5. Himachal Pradesh – covers the state of Himachal Pradesh
    6. Kolkata – covers Kolkata city
    7. Madhya Pradesh – covers the states of Madhya Pradesh & Chhattisgarh
    8. Odisha – covers the state of Odisha
    9. Rajasthan – covers the state of Rajasthan
    10. UP East – covers the state of Uttarakhand and eastern Uttar Pradesh
    11. UP West – covers Western Uttar Pradesh
    12. West Bengal – covers the state of West Bengal (except Kolkata)
  3. Mobile number – the number that made the call to Mobile Academy service
  4. Last called date – the date when the last call was made

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Under "Reports Category" select "Mobile Academy Reports", then




3) Under "Reports" select "Circle wise Anonymous Users"




4) Then select the Month for which the reports are needed




5) Then select respective Circle from "Circle" dropdown:




6) To view the result, click the "Submit" button.




7) Click on the download link to download the report. The selected report will be downloaded in Excel files.




8) Clicking on ‘Reset’ button will reset all the fields to their default values.


Cumulative Inactive Users


This report gives the line-listing of all such ASHAs who have not yet started the course as on date. The State / UT can act on this report and contact the ASHAs and motivate them to access the Mobile Academy course and complete the course. In this way we can bring a higher percentage of ASHAs accessing and completing the Mobile Academy program. This report is a very useful actionable report for the State / UT to monitor the ASHAs accessing the Mobile Academy course.

This line listing report would be available for download for the respective district and block users. These reports would be e-mailed to the respective block and district users on the day after it was generated. A summary mail giving the district wise break-up of the count of ASHA would be sent to the State / UT users on the same day.

Asha Rejected Records


This report gives the line-listing of all beneficiaries whose registered mobile numbers are either incorrect or are duplicate. These mobile numbers had flown in to the IVRS system but were rejected by the acceptance logic run by the IVRS system. The report contains the MCTS or RCH IDs. This report would be generated once every week preferably on Sunday. This line listing report would be available for download for the respective district and block users. These reports would be e-mailed to the respective block and district users on the day after it was generated –on Tuesday. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day. The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries and register their correct mobile numbers in the RCH Application. In this way we can correct the mobile numbers of beneficiaries and bring a higher percentage of beneficiaries under the ambit of Kilkari program. This report is a very useful actionable report for the State / UT to correct the beneficiary contact mobile number.

Steps to download the reports

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Under "Reports Category" select "Mobile Academy Reports", then




3) Under "Reports" select "Asha Rejected Records"




4.a) Then select the Month for which the reports are needed




4.b)Then select the 1st day of week(Sunday) for which the report is needed under "Select Week":




5) Then select an option in the 'State' dropdown for a specific state or to get all states data, select 'All':




6) Next, select an option from the 'District' dropdown in the same manner: (does not work if State is set to all)




7) Next, select an option from the 'Block' dropdown in the same manner: (does not work if District is set to all)




8) To view the result, click the "Submit" button.




9) Click on the download link to download the report. The selected report will be downloaded in Excel files.




10) Clicking on ‘Reset’ button will reset all the fields to their default values.

Asha Rejected Records


This report gives the line-listing of all beneficiaries whose registered mobile numbers are either incorrect or are duplicate. These mobile numbers had flown in to the IVRS system but were rejected by the acceptance logic run by the IVRS system. The report contains the MCTS or RCH IDs. This report would be generated once every week preferably on Sunday. This line listing report would be available for download for the respective district and block users. These reports would be e-mailed to the respective block and district users on the day after it was generated –on Tuesday. A summary mail giving the district wise break-up of the count of beneficiaries would be sent to the State / UT users on the same day. The State / UT can act upon this report and contact the ASHAs in the field and ask them to contact the said beneficiaries and register their correct mobile numbers in the RCH Application. In this way we can correct the mobile numbers of beneficiaries and bring a higher percentage of beneficiaries under the ambit of Kilkari program. This report is a very useful actionable report for the State / UT to correct the beneficiary contact mobile number.
