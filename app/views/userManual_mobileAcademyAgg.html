MA Cumulative Summary Report


This report provides the performance of the Mobile Academy project from the launch date of the service in the state to till date. It gives information about the number of ASHAs registered with the program, the number of ASHAs that have started the course, the number of ASHAs who have not yet started the course, the number of ASHAs who have passed the course and the number of ASHAs who have failed the course. All the above counts are for ASHAs with job status (GF status) ‘ACTIVE’.

This report will help to represent the overall performance of course completion by ASHA in selected state.

The report displays the data in the following sequence of columns.

  1. Serial Number
  2. State
  3. No of registered ASHAs
  4. No of ASHAs started course
  5. No of ASHAs not started course
  6. No of ASHAs successfully completed the course
  7. No of ASHAs who failed the course
  8. % not started course
  9. % successfully completed
  10. % failed the course

The rules governing the display components of the report is given below

  1. Columns 6 & 9 take into consideration only the first successful completion and not repeat attempts.
  2. The percentages displayed in the report follow the below rules based on data in its respective rows only
    1. % Not Started Course: (Column 5/Column 3) * 100
    2. % Successfully Completed: (Column 6/Column 4) * 100
    3. % Failed the course: (Column 7/Column 4) * 100

The report displays data based on the As-On date selected while downloading the report

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Under "Reports Category" select "Mobile Academy Reports".




3) Under "Reports" select "Cummulative Summary"




4) Select an ‘End Date’ for which the cumulative summary report is required.




5) Click on ‘Submit’




6) The report will be generated




7) Once the report is generated, it can be downloaded in PDF, CSV and XLSX formats.




MA Subscriber Report


This report provides a view of the data volume on the Mobile Academy service. It shows how many ASHAs who have registered but not completed the course at the start of the period, the number of ASHAs who have joined the course and number of who have completed the course. All the above counts are for ASHAs with job status (GF status) ‘ACTIVE’.

The report displays the data in the following sequence of columns

  1. Serial Number
  2. State
  3. No. of ASHA registered but not completed (period start)
  4. No. of ASHA records received through web service
  5. No. of ASHA records rejected
  6. No. of ASHA records added
  7. No. of ASHA successfully completed the course
  8. No. of ASHA registered but not completed (period end)

The rules governing the display components of the report is given below

  1. The data pertaining to column 3, 4, 5, 6, 7, 8 should be for the selected period only.
    1. Column 3 displays the number of ASHAs who have registered in the MA course prior to the start of the selected period but have not completed the course.
    2. Column 4 displays the number of ASHA Records that have been received from web service from MCTS/RCH during the selected period.
    3. Column 5 displays the number of the records that have been rejected due to wrong mobile number and other reasons in the selected period.
    4. Column 6 displays the number of ASHAs who have been added/subscribed in MA course for the first time in the selected period.
    5. Column 7 displays the number of ASHAs who have successfully completed the course for the first time, during the selected period and secured pass marks.
    6. Column 8 displays the number of ASHAs presently subscribed in the Mobile Academy program but are yet to start the course.

1) The users can login into the system using the URL: https://rchivrreports.in/.

2) Under "Reports Category" select "Mobile Academy Reports", then




3) Under "Reports" select "Subscriber Report"




4) Select ‘Year’ under ‘Period Type’.




5) Select the year for which report is needed.




6) Click on ‘Submit’.




7) The report will be generated.




8) Once report is generated, it can be downloaded in PDF, CSV or XLSX formats.




9) Select ‘Financial Year’ under ‘Period Type’ and choose the start year for which report is needed.




10) Click on ‘Submit’ to generate the report




11) The report will be generated which can be downloaded in PDF, CSV or XLSX formats.

12) Select ‘Quarter’ in the ‘Period Type’ and select the year and quarter for which the report is needed. Then click ‘Submit’.




13) The generated report can be downloaded in PDF, CSV and XLSX formats.

14) Select ‘Month’ under ‘Period type’ and choose a month and click ‘Submit’.




15) The report will be generated for the chosen month and can be downloaded in PDF, CSV and XLSX formats.

16) ‘Custom Range’ under ‘Period Type’ can also be selected. After that, you need to select a start date and an end date for which report is needed, then click on ‘Submit’




17) The report will be generated for the specified date range, which can be downloaded in PDF, CSV and XLSX formats.

MA Performance Report


This report provides the information about how many ASHAs have started the course in the selected period, number of ASHAs who have accessed the course during the period, number of ASHAs who had started the course previously but have not accessed the course during the selected period, number of ASHAs who have completed the course successfully in the selected period and those who have completed the course in the selected period but could not secure pass marks. All the above counts are for ASHAs with job status (GF status) ‘ACTIVE’.

The report displays the data in the following sequence of columns

  1. Serial Number
  2. State
  3. No of ASHA started course
  4. No of ASHA pursuing course
  5. No of ASHA not pursuing course
  6. No of ASHA successfully completed the course
  7. No of ASHA who failed the course

The rules governing the display components of the report is given below

  1. The data pertaining to column 3, 4, 5, 6 and 7 is for the selected period only.
    1. Column 3 displays the number of ASHAs who have started the course for the first time in the selected period.
    2. Column 4 displays the number of ASHAs who had started the course before the selected period, had accessed the course at least once with one bookmark during the selected period but not completed the course till the end of the selected period
    3. Column 5 displays the number of ASHAs who had started the course before the selected period but had not accessed the course even once during the selected period. The count does NOT include ASHAs who have any successful completion till the end of the selected period.
    4. Column 6 displays the number of ASHAs who have successfully completed the course for first time, during the selected period and secured pass marks.
    5. Column 7 displays the number of ASHAs who have completed the course during the selected period but did not secure passing marks even once till the end of selected period.
